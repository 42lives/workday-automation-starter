# Security

Do not report or commit real private work data to this repository.

This project is built around sample data and local dry-runs. If you find a workflow that could expose private files, emails, tokens, or personal information, open an issue with sanitized reproduction steps.

Recommended practice:

- use fake sample data,
- review outputs before sharing,
- keep real Gmail and Notion exports out of git,
- add privacy checks before connecting real services.
